# 📱 KeenKeeper

**KeenKeeper** is a smart relationship management tool. It helps you stay in touch with your friends and reminds you when it's time to reach out, ensuring no connection is lost.

---

## 🚀 Live Demo
[**View Live Project**](https://keen-keeper-phi-swart.vercel.app/)

---

## ✨ Key Features
* **Dashboard:** A quick overview of your friendship statuses.
* **Status Tracking:** Automatically marks friends as **Overdue**, **Almost Due**, or **On-Track**.
* **Quick Check-In:** Easily log calls, texts, or video chats.
* **History:** A timeline of all your past interactions.
* **Analytics:** Visual charts showing how you communicate with your friends.

---

## 🛠️ Tech Stack
* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS & DaisyUI
* **Icons:** React Icons
* **Charts:** Recharts
* **Deployment:** Vercel

---

## ⚙️ Local Setup
1. Clone the repo: `git clone https://github.com/Likhon-25/Keen-Keeper.git`
2. Install dependencies: `npm install`
3. Run the project: `npm run dev`
